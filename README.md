## 🚀 Featured Projects

### 🗓️ AI Appointment Scheduling Agent
A conversational scheduling agent built in n8n using **Gemini 2.5 Flash** and the **Google Calendar freeBusy API**. Handles OAuth2 authentication, real-time availability checks, and timezone-aware booking logic.

### 🧩 Multi-Agent Social Content Pipeline
A **52-node n8n workflow** orchestrating content generation, review, and publishing — integrating an LLM for creative generation, automated video generation, and Google Drive/Sheets for asset and data management end-to-end.

### ✅ Human-in-the-Loop Approval Workflow
A Slack-integrated review system using interactive **Block Kit** components with dynamic task IDs, routing AI-generated output through human approval before it ever ships.

### 🌦️ Weather Intelligence Agent
A two-tier **orchestrator/specialist** architecture — one workflow routes requests, a second handles live weather data (OpenWeatherMap) and LLM-based reasoning to generate context-aware responses.

### 📥 RAG Document Chatbot
Retrieval-augmented generation pipeline combining **Google Drive**, **Gemini embeddings**, and a **Pinecone vector store** for accurate, source-grounded document Q&A.

### 📧 Email Classification & Routing Agent
An automated triage system that classifies incoming email intent and routes it to the correct workflow or team — no manual sorting required.
